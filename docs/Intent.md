### Goal with the project

The point of the project is to have fully functioning advanced automation through the website generated by the automation web api.

There should be 3 modes of automation which you should be able choose when making a rule:
Simple, just select inputs and outputs with some simple functions like timer, rs-latch, etc.

FBD flow schematic, a fully interactive flow diagram, where you can do automation, with all the tools you could want for FBD programming.

Coding, you should be able to program the automation entirely through code.
